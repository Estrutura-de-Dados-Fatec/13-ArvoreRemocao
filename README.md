# Arvores (remoção de  elemento)
---

## Objetivos

Ao concluir esta atividade você deverá estender sua compreensão os seguintes conceitos:
* Remocao de elemento da arvore


## Atividade Proposta

Faça um fork deste repositorio e realize as seguintes atividades: 

- [ ] Complemente a função removerElementoArvore Arvore que deve remover um elemento da arvore mantendo suas propriedades. Lembre de tratar os 3 casos possiveis:  sem filhos, 1 filho, 2 filhos.
